# 34. Struts認定模試（100問）

## 目的

Apache Struts教材全体の理解度を確認するための模擬試験です。

## 出題範囲

- Struts概要
- Eclipse環境構築
- Action / JSP / struts.xml
- Strutsタグライブラリ
- Validation Framework
- Interceptor
- JDBC / ConnectionPool
- PostgreSQL
- DI
- JUnit / Mockito
- ログ
- 認証・認可
- 例外処理
- ファイルアップロード
- Ajax / REST API
- Spring連携
- Spring Boot移行
- セキュリティ
- 大規模開発

## 模試ルール

- 全100問
- 4択形式
- 制限時間: 90分
- 合格目安: 70問以上正解

## サンプル問題

### 問1

StrutsにおいてControllerに近い役割を持つものはどれか。

A. JSP  
B. Action  
C. CSS  
D. SQL  

答え: B

### 問2

Actionに直接書くべきでない処理はどれか。

A. Service呼び出し  
B. 遷移結果の返却  
C. SQL実行処理  
D. 画面表示用データの保持  

答え: C

### 問3

SQLインジェクション対策として適切なものはどれか。

A. SQLを文字列連結で作成する  
B. PreparedStatementを使用する  
C. JSPで入力値を隠す  
D. エラーを握りつぶす  

答え: B
